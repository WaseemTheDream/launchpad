## [1.0.0] - 2026-01-18

### Added
- Complete Launchpad Showcase app with 4 main tabs: Home, Guides, Commands, About
- Comprehensive command documentation with architecture details for all 13 commands
- Interactive guide system for learning Claude Code workflow
- Command detail view with Overview, Architecture, and Source tabs
- Bundled command source files viewable in-app
- Dark/Light theme support with Settings screen
- Material Design 3 UI with smooth navigation animations

### Technical
- Jetpack Compose UI with Material 3
- Navigation Compose for screen routing
- Asset-bundled markdown command documentation
- Release signing configured
